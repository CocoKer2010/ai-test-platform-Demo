# ai-test-platform-Demo
This project demonstrates how AI can be integrated into testing workflows to improve efficiency and scalability.

End-to-end AI-powered testing platform  built with FastAPI and LLM integration, covering test case generation, data generation, and automation execution

## ✨ Features
- AI-based test case generation
- AI-generated test data
- API automation with pytest
- FastAPI service endpoints

## 🧱 Architecture
Requirement → Test Cases → Test Data → Execution → Results

## 🚀 Quick Start

```bash
pip install -r requirements.txt
uvicorn app:app --reload
``


## Open:
http://127.0.0.1:8000/docs

## 🧪 Run Tests

```bash
pytest tests/
``
